# M3U php parser 
Изобретаю велосипед! :)

Парсер m3u плейлистов на php с использованием машины состояний (конечных автоматов) (FSM).   

Схема смены состояний:

![Схема смены состояний](/resources/statemachine.png "Схема смены состояний")

На данный момент поддерживаются следующие теги и аттрибуты:

 - [x] EXTM3U 
 - [x] EXTGRP 
 - [x] EXTINF
   - [x] group-title 
 - [x] EXTTV 
 - [x] COMMENTS
